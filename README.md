# Leet Battle

Two-player weekly DSA battle built on Striver A2Z problems with LeetCode-verified solves, MongoDB backend, and React/Vite frontend.

## Quick start
- Backend: `cd backend && pip install -r requirements.txt && cp .env.example .env && ./venv/bin/python manage.py runserver`
- Frontend: `cd frontend && npm install && cp .env.example .env && npm run dev`

## Key features
- Mongo-compatible auth backend with JWT stringified ObjectIds
- Daily assignments and verified solves against LeetCode submissions
- Weekly scoring, locking, and payments workflow
- Striver-ordered problem seed with AI pattern fill-ins
